# Assets Directory

This directory contains application assets for the Android APK.

## Required Files

### Application Icon
- **icon.png** - Application launcher icon (512x512 px recommended)
- **presplash.png** - Splash screen image (1024x1024 px recommended)

## How to Create Icons

1. Use an image editor to create a square PNG image
2. For icon.png: 512x512 pixels minimum
3. For presplash.png: 1024x1024 pixels minimum
4. Place them in this directory

## Placeholder Icons

Currently, placeholder files are needed. You can:
1. Use any image editor to create simple square images
2. Use online tools like https://www.favicon-generator.org/
3. Or modify buildozer.spec to comment out icon.filename and presplash.filename
